Convert vcards to FRITZ!Box address book files.

If you want to convert a vcard file into the format of a FRITZ!Box address book import file, you can use the vcard2fritzXML tool. It's written in Java, works on Mac OS X, Windows and Linux.
And it is OpenSource.

Learn more

vcard2FritzXML Get your FRITZ!Box synchronised.

vcard2fritzXML is a tool written in Java to convert a vcard file to a FRITZ!Box address book import file. It's under the GPLv2 license. More informations is on https://github.com/berkholz/vcard2fritzXML.
FRITZ!Box is a popular DSL router in Germany by AVM.
